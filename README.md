# Alby installer for Linux

This installer configures Alby for your browser on Linux and allows your to connect Alby to nodes behind Tor.

## Chrome

Open the terminal in this directory and run:

    $ ./chrome.sh

## Firefox

Open the terminal in this directory and run:

    $ ./firefox.sh

## Brave

Open the terminal in this directory and run:

    $ ./brave.sh

## Chromium

Open the terminal in this directory and run:

    $ ./chromium.sh
